== Description ==

Add subscibribers via the WooCommerce checkout to your Moosend mailing list.

Features:

*   Add or update subscribers via the checkout page
*   Option to select/enter name of list


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/woo-moosend` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerce->Settings->Integration screen to configure the plugin
